1. Input Data Format

EINVis requires two files: Nodes file and Edges file.

EIN Nodes file: Each line contains the information for one SNP. Each line contains four tokens: chromosome id, gene name, SNPs name and single-locus test statistic. The four tokens are separated by space symbol " ". The SNPs are ordered by their positions on the chromosomes. One example Nodes file is shown as follows:

Line #chrom_id gene_name SNP_name single-locus_test_statistic
11 CA6 rs7513804 0.382463
22 IL1A rs1894399 0.712872
37 MTPN rs1488004 0.477534
411 CD3E rs3782040 0.490576
521 CSTB rs743466 0.12148
622 TIMP3 rs130293 0.889259

EIN Edges file: Each line contains a SNP-pair and its two-locus test statistic. Each line contains three tokens: Index of the First SNP, Index of Second SNP, and Two-Locus Test Statistic. The "Index of SNP" means the index in the list of SNPs in the "Nodes file". The index begins from "0". The three tokens are separated by space symbol " ". The SNP-pairs are sorted in descending order by their two-locus test statistics. One example of this file is shown as follows:

Line #SNP1-Index SNP2-Index two-locus_test_statistic
15 1 1
22 0 0.895698
32 3 0.741046
44 1 0.638908
50 5 0.000631648
63 4 0

From this example, we can see that the first line "5 1 1" means that SNPs "22 TIMP3 rs130293" (the 6th line of the table for the Node file) and "2 IL1A rs1894399" (the 2nd line of the table for the Node file) have an interaction with the test value 1.

The single-locus test value should be between 0 and 1. We recommend that the single-locus test value is normalized among the single-locus test values of all the SNPs. Two-locus test values also should be between 0 and 1. The user can select the top ranked SNP-pairs, e.g., the top-400 SNP-pairs. Then the two-locus test values should be normalized among the selected SNP-pairs.

LD Edges file: Each line contains a SNP-pair and its LD value. Each line contains three tokens: Index of the First SNP, Index of Second SNP, and LD value. The "Index of SNP" is the same as in "EIN Edge file", and it means the index in the list of SNPs in the "Nodes file". The index begins from "0". The three tokens are separated by space symbol " ". One example of this file is shown as follows:

Line #SNP1-Index SNP2-Index LD-value
10 4 0.5
22 1 0.495
31 3 1.0
44 2 0.923
50 2 0.054
61 4 0.1

The "LD value" can be a number between 0 and 1, for example, the r^2 statistic.

2. User Interactions

EINVis provides controls over the EIN tree ring view bubble, e.g. slider, to let user to adjust the visualization effect. EINVis also provides mouse user interactions combined with keys pressed to select, expand or collapse the nodes.

The control functions in the EIN tree ring view are listed in the following table. You can find these controls in the controls panel (right column) in EIN tree ring view bubble.

Control TypeControl NameFunction
ButtonChoose File (Nodes File)Choose the EIN Nodes File
Choose File (Edges File)Choose the EIN Edges File
Choose File (LD Edges File)Choose the LD Edges File
Start RenderingStart to render the views after user choose the Nodes File and Edges File
DemoStart to render the demo. EINVis will load the demo Nodes File and Edges File automatically from server.
View RankOpen a new webpage to show the table of rank of SNPs based on their degree.
SliderEdge Number Control how many edges are shown in tree ring views and matrix view.
Minimum: 10%, Maximum: 100% (All the edges), Delta: 10%.
Edge Bundle Control the edge bundle strength.
Minimum: 0.0 (Straight Line), Maximum: 1.00, Delta: 0.1.
Font Size Control the font size of the node name text.
Minimum: "10 px", Maximum: "20 px", Delta: "1 px".
When user decrease the font size, the width of three levels in the tree ring views will also decrease.
CheckboxNode NameControl whether or not show the node name text
Black BackgroundControl whether use black background in tree ring views. If checked, render black background, it will be much clear; if unchecked, render white background.
Drop-Down MenuNode Color Bar Select one Node Color bar from the drop-down menu, the contents are listed in next Table.
Edge Color Bar Select one Edge Color bar from the drop-down menu, the contents are listed in next Table.
RadioHighlight Any If user switch to this radio button, and user has selected multiple nodes, then nodes related to any of the selected nodes and induced edges will be highlighted with color.
Highlight All If user switch to this radio button, and user has selected multiple nodes, then nodes related to all of the selected nodes and induced edges will be highlighted with color.
Input Text BoxSearch Input the Gene name or SNPs ID in this text box, then, EINVis will highlight all the matched Genes or SNPs.
